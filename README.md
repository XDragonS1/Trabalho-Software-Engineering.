TaskFlow - Sistema Web de Gerenciamento de Tarefas

Objetivo
Desenvolver um sistema para cadastro, acompanhamento e gerenciamento de tarefas utilizando conceitos de Engenharia de Software e metodologias ágeis.

Escopo Inicial
O sistema permite:
Criar tarefas
Editar tarefas
Excluir tarefas
Visualizar tarefas

Metodologia
Foi utilizada a metodologia Kanban através do GitHub Projects.

Como executar
pip install -r requirements.txt
python src/app.py

Como executar os testes
pytest
